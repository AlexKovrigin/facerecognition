# facerecognition
Python cctv-like project

## About project
Project's idea is to use standard, connected to desktop webcam as cctv camera, which also has face recognition built in!

## External usage
Not really intended, but doable

## Dependencies
* dlib
* json
* os 
* random 
* string
* datetime
* glob
* cv2
* time
* threading
* numpy
* multiprocessing
* skimage
* scipy.spatial
